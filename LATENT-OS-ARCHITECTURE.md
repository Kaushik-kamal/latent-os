# LATENT OS Architecture

## Overview
LATENT OS is a Startup Operating System built inside Notion.

## Core Modules
- Home Dashboard
- Task Manager (Kanban Board)
- Build Log
- Daily Standup
- AI Meeting Summary
- Team Dashboard

## Tech Stack
- Notion (Core Platform)
- Make.com (GitHub Webhook Automation)
- Notion AI (Meeting Summaries)
- GitHub (Version Control)
